# Blog

This is a simple blog platform that can be used to practice Gin+Gorm.



**Tech Stack**

- Frontend:
  - Bootstrap
  - mdeditor

- Backend:
  - Gin
  - Gorm
  - Postman

**Feature**

- Register and login
- Post blog
- Read blog posted by other users
